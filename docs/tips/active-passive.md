# 能動態と受動態を使い分ける
## 説明
文章の統一性を保つため、能動態と受動態を使い分けます。

- 能動態：主語が行為を行っている場合に使用
    - 例：メッセージを表示**します**
- 受動態：主語が好意を受けている場合に使用
    - 例：メッセージが表示**されます**

## 利用例(能動態)
**NG:**

ログインボタンがクリックされます。

**OK：**

ログインボタンをクリックします。

## 利用例(受動態)
**NG:**

ログインに失敗すると、エラーメッセージを表示します。

**OK：**

ログインに失敗すると、エラーメッセージが表示されます。
